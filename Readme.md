## Seeker

- It seems like tracking model aircraft, or drones, visually, especially from the ground in daylight conditions is trivially easy.   
- Pursing the drone at speed while seeking to null the lateral velocity should be enough to "catch the snitch."  
- So, this software aims to be a seeker of rc aircraft that is lightweight enough to run on any RPI 5.
  - Ambitiously, I'd prefer it to run on a zero 2 w.  

### Everything Is Trivial

- I literally got the idea to do this fom James May's off handed comments about being on a collision course with a ship because they were on a "constant bearing" 
- This is a smooth brained project.
